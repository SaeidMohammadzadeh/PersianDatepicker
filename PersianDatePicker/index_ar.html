Persian Datepicker — عرض تجريبي

مكوّن تقويم خفيف يدعم ثلاثة أوضاع: جلالي، ميلادي، قمري. مصمّم للاستمارات وتطبيقات SPA.

المعاينة والاستخدام انقر على أي حقل لفتح التقويم — اضغط Esc للإغلاق

مستندات سريعة

فئات الحقول

.date-input-field — جلالي
.gdate-input-field — ميلادي
.hdate-input-field — قمري

حقل مخفي

ضع input[type="hidden"] مباشرة بعد الحقل الظاهر لاستقبال القيمة. الدوال window.syncJalaliDate, window.syncGregorianDate, window.syncHijriDate تقوم بتعيين القيمة.
التثبيت
  1. أضف css/persiandatepicker.css و js/persiandatepicker.js إلى صفحتك.
  2. أدرج الخط المناسب (اختياري).
  3. حدد الحقول بواسطة الفئات أعلاه.
HTML المطلوب 
<input class="date-input-field" placeholder="Select date">
<input type="hidden" name="jalaliDate">
سهولة الوصول
  • اضغط Escape لإغلاق التقويم.
  • القيمة المختارة تُكتب في الحقل المخفي ويتم إصدار حدث input.
تهيئة يدوية (اختياري) 
<!-- يتم التهيئة تلقائياً عند DOMContentLoaded; استدعِ init يدوياً إذا احتجت -->
<script>
  if (typeof initJalaliDatepicker === 'function') {
    initJalaliDatepicker();
  }
</script>

ملاحظات فنية

منطق العطل: تُعرَّف العطل كسلاسل جلالية؛ خلايا الميلادي والقمري تؤشّر بتحويل التاريخ للجلالي ومقارنتهما (رسمي = أحمر، غير رسمي = برتقالي).

منع الوميض: تُستخدم أحداث pointerdown وعلم مؤقت لمنع وميض النقر الأول.

  • ميلادي: الاثنين .. الأحد (الأحد العمود الأخير)
  • قمري: السبت .. الجمعة (الجمعة العمود الأخير)
